## Responsible Disclosure

If you discover a vulnerability or security issue:
1. Email us at: **ankitkumarforall@gmail.com**  
2. Provide detailed information about the issue, including steps to reproduce it.
3. Do not publicly disclose the vulnerability until it is resolved.

We will acknowledge reports promptly and work to fix issues responsibly.

---

## Security Principles

- **Privacy-First Design:**  
  No personally identifiable information (PII) is collected; all predictions are zone-based.

- **Encrypted Communication:**  
  All frontend-backend interactions use secure HTTPS connections.

- **Access Control:**  
  API endpoints are authenticated and validated to prevent unauthorized access.

- **AI Model Integrity:**  
  Machine learning models are sandboxed and monitored to prevent tampering.

- **Data Handling:**  
  Historical and synthetic datasets are anonymized and sanitized.

- **Dependency Management:**  
  Third-party libraries are regularly audited, patched, and updated.

- **Safe Deployment Practices:**  
  Cloud infrastructure and mobile deployments follow standard security protocols.
